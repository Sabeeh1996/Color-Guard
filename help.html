Accessibility Mode Settings

High-Contrast Level

Range: 1.0x - 3.0x (default: 2.5x)

  • 1.0x - Normal contrast (no change)
  • 1.5x - 2.0x - Mild enhancement
  • 2.5x - 3.0x - Strong enhancement for low vision
üí° Tip: Start with 2.0x and adjust based on your comfort level.

Hue Shift Amount

Range: 0¬∞ - 180¬∞ (default: 90¬∞)

  • 30¬∞ - 60¬∞ - Subtle color adjustment
  • 90¬∞ - 120¬∞ - Recommended for most users
  • 150¬∞ - 180¬∞ - Maximum color shift

Edge Outline Thickness

Range: 1px - 5px (default: 2px)

  • 1px - Subtle outlines
  • 2px - 3px - Balanced visibility
  • 4px - 5px - Maximum emphasis

üîß Troubleshooting

Extension doesn't work on some pages

Problem: Content Security Policy (CSP) restrictions block the extension

Affected pages:

  • Chrome internal pages (chrome://)
  • Chrome Web Store
  • Some banking/security-focused sites

Solution: This is a browser security feature and cannot be bypassed. The extension will work on other pages.

No effect after enabling a mode

Checklist:

  1. Verify mode is not set to "Off" in the popup
  2. Check if current site is blacklisted (Options page)
  3. Reload the extension in chrome://extensions
  4. Refresh the webpage
  5. Check browser console (F12) for error messages
Settings not saving

Solutions:

  • Make sure you clicked "Save All Settings" button
  • Verify Chrome sync is enabled in Chrome settings
  • Check if you have sufficient storage quota
  • Try disabling and re-enabling the extension
Page becomes slow or unresponsive

If you experience performance issues:

  • Edge-Highlight mode is most resource-intensive
  • Try reducing outline thickness to 1px
  • Switch to High-Contrast or Hue-Shift mode
  • Add slow pages to blacklist
  • Close unused tabs to free up memory
Colors look wrong or strange

For Hue-Shift mode:

  • Try different hue amounts (30¬∞, 60¬∞, 90¬∞, 120¬∞)
  • Combine with High-Contrast for better results
  • Some color combinations may look unusual - this is normal

For High-Contrast mode:

  • Very high values (>2.5x) may cause color clipping
  • Reduce contrast level if colors look oversaturated

‚ùì Frequently Asked Questions

Is ColorGuard free?

Yes! ColorGuard is 100% free and open-source under the MIT license. You can use it, modify it, and even contribute to its development.

Does ColorGuard collect my data?

No. ColorGuard processes everything locally on your device. No data is sent to external servers, and we don't collect any usage information. See our Privacy Policy for details.

Will ColorGuard slow down my browsing?

No. ColorGuard uses GPU-accelerated CSS filters for High-Contrast and Hue-Shift modes, which have minimal performance impact. Edge-Highlight mode may use slightly more resources on pages with many interactive elements.

Can I use ColorGuard on mobile?

Currently, ColorGuard is only available for desktop Chrome browsers. Mobile browser support may be added in future versions.

Does ColorGuard work with other accessibility tools?

Yes! ColorGuard is designed to work alongside screen readers, browser zoom, and other accessibility features. However, some combinations may produce unexpected results - test what works best for you.

Which color blindness types does ColorGuard support?

ColorGuard's Hue-Shift mode is primarily designed for red-green color blindness (deuteranopia and protanopia), which affects about 8% of males and 0.5% of females. Future versions may include presets for other types like tritanopia (blue-yellow).

Can I use different settings for different websites?

Currently, ColorGuard uses global settings for all websites (with whitelist/blacklist for enabling/disabling). Per-site custom settings may be added in a future version.

How do I report bugs or request features?

Visit our GitHub Issues page to report bugs or request features. Please include:

  • Browser version and operating system
  • Steps to reproduce the issue
  • Screenshots if applicable
  • Console error messages (F12 ‚Üí Console)
Can I contribute to ColorGuard?

Absolutely! ColorGuard is open-source and welcomes contributions. Check out our Contributing Guide to get started.
