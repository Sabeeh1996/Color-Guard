Frequently Asked Questions

Is ColorGuard free?

Yes! ColorGuard is 100% free and open-source under the MIT license.

Does it work on all websites?

ColorGuard works on most websites. It cannot run on Chrome internal pages (chrome://) or some sites with strict security policies.

Will it slow down my browsing?

No! ColorGuard uses GPU-accelerated CSS filters with minimal performance impact.

Is my data safe?

Absolutely! All processing happens locally on your device. No data is sent to external servers.

Can I customize the settings?

Yes! Adjust contrast levels, hue rotation, outline thickness, and create per-site rules.

How do I get help?

Check our documentation or open an issue on GitHub.
